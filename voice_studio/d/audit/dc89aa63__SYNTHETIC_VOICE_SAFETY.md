# Synthetic Voice Safety Notes

Voice Studio is intended for fictional D&D and tabletop characters. Do not use it to impersonate real people, bypass consent, or mislead listeners about who is speaking.

Recommended table practice:

1. Keep generated files labeled with character name and synthetic status.
2. Use only source samples you created, own, or have permission to use.
3. Do not name a profile after a real person unless you have permission and a legitimate reason.
4. Store profile JSON beside exported audio so the generation settings are auditable.
