Singular Accent Atlas split archive

This atlas was split into 2 independent ZIP files because the full archive was too large.

How to restore:
1. Download BOTH ZIP files.
2. Extract Part 1 into a folder of your choice.
3. Extract Part 2 into the SAME folder.
4. Allow folders to merge if Windows asks.

Both ZIPs contain paths under Singular_Accent_Atlas/ so they merge back into one atlas folder.
No source content was intentionally removed in the split.
