# Belavadös Voice Filter Notes

This build applies biome first, then race/category texture, then class/subclass behavior, then gender-expression overlay. Gender identity is treated as expression/identity metadata, not a biological voice rule.

## Biome accent carriers

- **Ocean Surface Floating Settlement** → **Tidecrest Cant** (Portuguese): Open, rolling, maritime, lively, salt-air social.
- **Underwater With Reefs** → **Reefglass Lilt** (Japanese): Graceful, clear, flowing, elegant, refracted like light through water.
- **Underwater Without Reefs** → **Deepcurrent Song** (Thai): Quiet, deep, rhythmic, mysterious, pressure-heavy.
- **Grassland** → **Plainwind Common** (American Midwest / General North American): Broad, open, easygoing, practical, approachable.
- **Prairie** → **Ironstep Cant** (Russian): Strong, wide, grounded, hardy, martial when needed.
- **Farming** → **Hearthfield Brogue** (Irish): Friendly, earthy, communal, homely, harvest-warm.
- **Mountain Range** → **Cragthane Burr** (Scottish): Tough, crisp, resilient, wind-carved.
- **Valley** → **Vinesong Flow** (Italian): Flowing, fertile, elegant, soft, emotionally expressive.
- **Deep Cavern** → **Stonehollow Echo** (Welsh): Echoing, ancient, carved-from-stone, heavy with memory.
- **Deep Forest** → **Rootmere Cant** (German): Dense, structured, rooted, authoritative, old-growth serious.
- **Partial Forest** → **Sundapple Tongue** (Spanish): Flexible, social, sun-dappled, active, borderland-friendly.
- **Treetops / Treehouses** → **Highbranch Lilt** (French): Light, elevated, refined, airy, graceful.
- **Marshes and Swamps** → **Mirecurl Drawl** (Louisiana Cajun / Creole-influenced English): Slow, earthy, wet, lived-in, haunted when death-touched.
- **Beach and Grass With Water** → **Brightshore Flow** (Brazilian Portuguese): Warm, coastal, bright, casual, breezy.
- **Beach and Reefs With Water** → **Wavebloom Welcome** (Hawaiian-influenced English): Gentle, tropical, wave-like, welcoming, communal.
- **Hybrid Tree and Forest Floor** → **Bramblewood Burr** (English West Country): Rustic, layered, grounded, woodland, friendly but sturdy.
- **Hybrid Farming Forest Grassland** → **Millfield Practical** (Dutch): Practical, mixed, industrious, balanced, workday-clear.
