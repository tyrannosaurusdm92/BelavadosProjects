# Licenses and Bundling Notes

This build bundles original scanner code and transformed JSON/profile data. It does **not** copy third-party repository source code into the runnable scanner modules.

The attached repositories and resource packs were treated as reference material for docs, manifests, speech-math naming, and optional future integration notes. Their own licenses and README files remain authoritative for any future direct reuse.

Large voice/audio packs are not bundled into the static scanner. Reference audio can be imported by the user at runtime as influence metadata only.

See `RESOURCE_INPUT_AUDIT.md` for archive-level hashes and detected README/LICENSE paths.
