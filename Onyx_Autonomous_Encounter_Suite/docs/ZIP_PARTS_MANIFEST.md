# Zip Parts Manifest

Extract all three zip files into the same folder. The zip files contain direct program paths; they do not contain nested zip archives.

- Part 1: `encounters_minigame.html`, `css/`, `js/`, `json/`, `data/`, `docs/`, and small/core assets.
- Part 2: dice bot tokens, Onyx moods, and media assets.
- Part 3: large FX and token asset libraries.

After extraction, open `encounters_minigame.html`.
DM tools are behind `#dm-session` or `#dm-editor`.
