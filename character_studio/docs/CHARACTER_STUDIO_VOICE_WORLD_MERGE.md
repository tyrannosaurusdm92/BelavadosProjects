# Character Studio Voice World Merge

The former voice atlas is integrated into Belavadös Character Studio as Page 3 Character Voice World and a Page 2 sheet voice summary strip. It keeps the project as a character-building site while expanding biome, race, class, crossover, slider, audio transport, and backend payload capability. Backend deployment details are not rendered in the player-facing UI.

- 17 canonical fantasy accent folders under `voice-models/main-accents/`
- 680 lightweight crossover folders under `voice-models/biome-crossovers/`
- No duplicate model binaries in crossover folders
- Full 27-control slider schema active on Page 3
- Weighted layer mixer with enable/disable and per-layer intensity
- One active media/speech clip at a time
- Pause, stop, rewind 5 seconds, and forward 5 seconds controls for audio and speech preview
