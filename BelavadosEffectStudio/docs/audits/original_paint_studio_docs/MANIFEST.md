# Project Manifest

## Runtime files

- `index.html` — Application shell and all controls.
- `css/styles.css` — Dark cyan Belavados editor styling, panels, checkerboard, responsive layout, GeoJSON anchor visuals.
- `js/app.js` — Canvas editor, transparency scanning, tools, selection system, GeoJSON anchor editor, exports, and browser save.

## Documentation

- `README.md` — User-facing run instructions and workflow.
- `docs/SOURCE_AUDIT.md` — How the uploaded repositories were merged conceptually.
- `docs/TEST_REPORT.md` — Validation notes.
- `docs/licenses/` — License texts found in the uploaded repository bundle.
- `docs/*_README_OR_NOTES.txt` — Preserved upstream readmes/notes for traceability.
