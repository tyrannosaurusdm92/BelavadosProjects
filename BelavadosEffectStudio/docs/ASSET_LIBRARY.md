# Belavadös Effect Studio v2 Asset Library

Bundled browser-friendly assets: **267**
Approximate extracted asset payload: **43.8 MB**

## Categories
- audio: 22
- effects: 22
- frame markers: 128
- nature: 49
- stickers: 12
- textures: 18
- ui stamps: 16

## What was included
- PNG/JPG/GIF image assets that can be stamped as editable overlays.
- WAV/MP3 audio assets exposed in the Assets panel for preview/reference.
- Frame-marker images, magic/effect sprites, textures, nature/terrain images, UI stamps, and sticker-like art.

## What was intentionally excluded
- Font files (`.ttf`, `.otf`, `.woff`, `.woff2`) were not bundled.
- Unity/editor-only files (`.prefab`, `.asset`, `.mat`, `.cs`, `.shader`, `.fbx`, `.meta`, `.unity`) were not bundled into the web app runtime.
- Very large texture/source files were skipped unless they were directly useful as browser stamp/effect assets.

## Runtime use
Open the **Assets** icon panel, choose a category, then press an asset tile to add it as an editable overlay. Overlays can be moved, resized, rotated, raised/lowered, duplicated, flipped, deleted, linked, or rasterized onto the active drawing layer.
