# Belavadös Effect Mobile Studio — Quickstart

1. Open `index.html` or install the PWA from your hosted GitHub/Apps Script page.
2. On mobile, keep **Safe Scroll On** while navigating, zooming, or scrolling.
3. Upload an image from **Files / Upload**.
4. Pinch inside the image box to zoom. Drag inside the image box to pan.
5. Open **Brush + Effect Creator**, choose the effect, color, size, and sliders.
6. Tap **Safe Scroll On** to switch to **Touch Drawing Armed** only when ready to paint.
7. Turn Safe Scroll back on after drawing so accidental finger movement does not edit the canvas.
8. Export as PNG, project JSON, or interactive HTML.

Portrait mode is the primary layout. Landscape/rotated view is supported with a shorter workspace and compact mobile dock.


## New lightning-safe mobile notes

The new lightning modes are deliberately still hidden inside the Brush + Effect Creator dropdown flow on phones. For fat-finger safety, leave Safe Scroll ON while navigating, then tap the canvas arm button only when you are ready to paint. Use Chain Lightning or Forked Lightning for weapon discharge, Storm Mesh for storm/lattice fields, Spark Field for crackle and mote bursts, and Ichor Discharge when you want fluid thundercoil current.
