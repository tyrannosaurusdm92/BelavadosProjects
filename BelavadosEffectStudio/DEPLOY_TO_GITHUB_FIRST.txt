This package is flattened for GitHub Pages. Upload the CONTENTS of this folder into your existing GitHub Pages folder, not the folder itself.

1. Desktop, tablet, mobile portrait, and rotated/landscape now use the SAME app shell.

Deploy steps:
1. In your GitHub repo, open the folder that GitHub Pages actually serves for this app.
2. Delete the old app files in that folder first: old index.html, old assets, old old alternate PWA folder, old sw.js, old manifest, and old icons if present.
3. Upload every file/folder from this deploy package directly into that same folder.
4. Commit changes.
5. Open github-cache-reset.html once from the live GitHub Pages URL. It clears old service workers/cache storage and redirects to index.html.
6. On Android, also remove the old home-screen icon and add the app again after the reset if it still opens the older cached layout.
